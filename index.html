Ayo! Cegah Kebakaran Sebelum Melanda

Meningkatnya jumlah kebakaran di Indonesia menyebabkan banyak orang menderita kerugian yang sangat besar, baik kerugian materil bahkan sampai merenggut nyawa. Namun sebenarnya kebakaran merupakan bencana yang dapat dicegah, berbeda dengan bencana alam, karena sebagian besar musibah kebakaran disebabkan oleh kelalaian dan kecerobohan manusia.

Berdasarkan fakta tersebut, PT. Mandiri Central Wahana (FireBlock) Indonesia ingin turut serta mengedukasi masyarakat Indonesia tentang pentingnya memiliki kesadaran akan bahaya kebakaran dan kami berusaha memberikan kemudahan dalam menjaga keselamatan orang dan barang.

Atas dasar hal ini, maka kami mengembangkan suatu produk inovatif, yaitu alat pemadam otomatis dengan teknologi terbaru "FireBlock" mempunyai kemampuan yang tidak dimiliki oleh alat pemadam api lain (Otomatis & Cepat).

OTOMATIS

FireBlock akan bekerja secara Otomatis tanpa bantuan tenaga manusia dan siaga 24 jam / 7 hari

CEPAT

FireBlock akan memadamkan api secara cepat dalam hitungan detik api mati total

SEMUA KELAS API

FireBlock dapat memadamkan di semua Kelas Api

TANPA PERAWATAN

Tidak perlu isi ulang selama 6 Tahun (FireBlock) & 3 Tahun (Power X)

RAMAH LINGKUNGAN

FireBlock memiliki kandungan yang Ramah Lingkungan dan Tidak berbahaya bagi Manusia

SERTIFIKASi

FireBlock telah memiliki sertifikasi Nasional dan Internasional

VISI FIREBLOCK

Menjadikan PT. Mandiri Central Wahana sebagai perusahaan alat pemadam yang berstandar Internasional

MISI FIREBLOCK

Kami membantu orang - orang merasa aman dan nyaman dalam menikmati kehidupan dengan terbebas dari bahaya kebakaran.

Kami akan menginspirasi orang - orang untuk melakukan cara baru yang lebih praktis, efesien dan efektif dalam memadamkan api dengan tidak menimbulkan dampak negatif terhadap lingkungan sekitar.

Kami Sepenuhnya menyadari bahwa kami akan selalu mengembangkan inovasi baru untuk pertumbuhan usaha.

Kami bertujuan melaksanakan program kerja jangka panjang dengan supplier kami, para pelanggan dan rekan lainnya dalam mencapai tujuan ini.

PERHATIAN!!

Wajib Ketahui Cara Kerja Efektif FireBlock

FireBlock termasuk didalam golongan Alat Pemadam Api Ringan (APAR)

FireBlock akan lebih efektif untuk pemadaman api tahap awal atau preventif (Pencegahan), dan bekerja secara maksimal saat diletakan/dipasang dititik rawan timbulnya api, sebelum meluasnya titik kebakaran

FireBlock dan pada umumnya golongan APAR tidak disarankan penggunaannya pada saat tahap pembakaran yang sudah meluas (Flash Over) atau kebakaran besar

Apabila api sudah membesar segera hubungi petugas pemadam kebakaran terdekat!

Penggunaan Otomatis

Tempatkan FireBlock dan Power X pada titik yang berpotensi terjadinya kebakaran (Panel Listrik, MCB, KwH Meter, Stop Kontak, Tabung Gas, Genset, dll) FireBlock dan Power X akan bekerja efektif dan cepat untuk memadamkan api, tanpa tenaga manusia

Penggunaan Manual

Apabila sudah terjadi kebakaran, FireBlock juga dapat digunakan secara Manual dengan cara Melemparkan FireBlock pada titik yang terjadi kebakaran
